<h2>ğŸŒŸ Key Features</h2>

<h3>Core Platform:</h3>
<ul>
  <li><strong>Authentication</strong>: Built-in auth with OAuth support</li>
  <li><strong>Real-Time Chats</strong>: Direct & group messaging</li>
  <li><strong>Prayer System</strong>: Quests, inventory, store & currency</li>
  <li><strong>Forum</strong>: Categorized topics & threaded posts</li>
  <li><strong>Admin Dashboards</strong>: Comprehensive management tools</li>
</ul>

<h3>User Experience:</h3>
<ul>
  <li><strong>Customization</strong>: Personalize site appearance and preferences</li>
  <li><strong>Notifications</strong>: Keep users updated with alerts and messages</li>
  <li><strong>Auto-Generating Avatars</strong>: Unique user icons for each member</li>
  <li><strong>Static Plots</strong>: Data visualization and insights dashboard</li>
</ul>

<h3>Moderation & Security:</h3>
<ul>
  <li><strong>Ban System</strong>: Robust access control and user management</li>
  <li><strong>Permission Controls</strong>: Granular access levels for different roles</li>
</ul>
